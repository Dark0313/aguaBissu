# Agua Bissu - Admin Panel

Este es el panel administrativo para la gestión de flota y clientes de **Agua Bissu**.

## 🚀 Inicio Rápido

1. **Instalación**:
   ```bash
   npm install
   ```

2. **Desarrollo**:
   ```bash
   npm run dev
   ```

3. **Producción**:
   ```bash
   npm run build
   ```

## 🛠️ Tecnologías Principales

- **React** (State management, Hooks).
- **Vite** (Build tool).
- **Lucide-React** (Iconos).
- **CSS Vanilla** (Estilos con variables).
